To run the application, navigate to the project directory and run the following commands:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run.py
```
The API will be available at `http://localhost:5000`. You can use a tool like `curl` or a REST client to test the API endpoints. For example:
```bash
curl -X GET http://localhost:5000/todos
curl -X POST -H "Content-Type: application/json" -d '{"title": "New Todo", "description": "This is a new todo"}' http://localhost:5000/todos
```
